# 🇦🇪 UAE Job Market Analysis 2026

An interactive dashboard analyzing the UAE Data Science, ML, AI & Data Analysis job market.

## 🔗 Live Demo
Open `index.html` directly in browser — no server needed!

## 📊 Features
- 41 real job listings from Dubai & Abu Dhabi
- Interactive filters by job category
- 7 charts: salary analysis, skills demand, heatmap
- Top hiring companies & salary ranges

## 🛠️ Tech Stack
Python | Streamlit | Plotly | Pandas | Chart.js | HTML/CSS/JS

## 🚀 Run Locally
pip install -r requirements.txt
streamlit run app.py
